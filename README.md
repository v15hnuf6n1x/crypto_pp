# Cryptocurrency Price Prediction - Multi-Model ML Pipeline

A comprehensive machine learning solution for predicting cryptocurrency prices using multiple state-of-the-art models and ensemble techniques.

## Features

- **Multiple ML Models**: ARIMA, LSTM, XGBoost, and Facebook Prophet
- **Automated Data Pipeline**: Fetch, preprocess, and feature engineering
- **Model Comparison**: Comprehensive evaluation and comparison metrics
- **Ensemble Prediction**: Weighted ensemble for improved accuracy
- **Technical Indicators**: 15+ technical indicators for feature engineering
- **Model Persistence**: Save and load trained models
- **Visualization**: Detailed performance comparison plots
